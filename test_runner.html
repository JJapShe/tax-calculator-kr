📋 테스트 실행 방법

1. 위의 버튼을 클릭하여 테스트를 실행하거나

2. 개발자 도구 (F12)를 열고 Console 탭에서 다음 명령어를 실행하세요:

  • runTests() - 전체 테스트 실행
  • testElectricCar() - 전기차 테스트
  • testDisabledExemption() - 장애인 면제 테스트
  • testCompactCar() - 경차 테스트

🔍 테스트 커버리지

  • ✅ 기본 승용차 세금 계산
  • ✅ 전기차 자동 감면 적용 (개별소비세 300만원, 취득세 140만원 한도)
  • ✅ 수소전기차 자동 감면 적용 (개별소비세 400만원, 취득세 140만원 한도)
  • ✅ 경차 자동 감면 적용 (개별소비세 면제, 취득세 1875만원 이하 면제)
  • ✅ 장애인 면제 적용 (개별소비세 500만원 한도, 취득세 면제)
  • ✅ 1000cc 이하 비과세 (전기차/수소전기차 제외)
  • ✅ 전기차 감면 한도 제한 적용
